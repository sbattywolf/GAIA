GAIA CLI (minimal)

Usage examples:

- Dry-run installer:

```powershell
python -m gaia.cli install
```

- Apply installer:

```powershell
python -m gaia.cli install --apply
```

- List agents:

```powershell
python -m gaia.cli agents list
```

- Start agent:

```powershell
python -m gaia.cli agents start --agent-id alby-online-0.3
```
