## Key Findings Summary

### üî¥ Critical Changes
1. **No automatic renewal** - Must apply even to maintain position
2. **24 CFU no longer valid** - New 60 CFU pathways required
3. **Tight deadline** - Only 20 days for application submission
4. **Certification restrictions** - Only Accredia-accredited IT certifications

### üü¢ Positive Changes
1. **Re-entry algorithm** - More meritocratic assignment system
2. **Teaching continuity** - Priority for support teacher confirmation
3. **Greater transparency** - Clearer criteria and predictable processes
