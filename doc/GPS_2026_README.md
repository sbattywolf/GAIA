## What is GPS?

GPS (Graduatorie Provinciali per Supplenze) is the Italian provincial ranking system for assigning substitute teaching positions in public schools. This analysis covers the major 2026 update affecting the 2026-2028 biennium.

## ‚ö° Critical Information

### üî¥ Most Important Changes

1. **No Automatic Renewal** - Must submit application even to maintain current position
2. **24 CFU No Longer Valid** - New 60 CFU pathways required
3. **20-Day Application Window** - Late January to mid-February 2026
4. **Certification Restrictions** - Only Accredia-accredited IT certifications

### ‚è∞ Critical Timeline

- **December 2025**: Verify all certifications
- **January 2026**: Prepare all documentation
- **Late Jan - Mid Feb 2026**: 20-day application window
- **Spring 2026**: GPS publication
- **Summer 2026**: Assignment begins
