## Introduction

The GPS (Graduatorie Provinciali per Supplenze - Provincial Rankings for Substitute Teachers) update for the 2026-2028 biennium introduces substantial changes to the teacher ranking and substitute assignment system. These modifications aim to make the system more meritocratic, transparent, and efficient, addressing critical issues that emerged in previous editions.

This document analyzes the main updates in detail, provides operational guidelines, and illustrates the expected impact on teachers.

---

## Main Updates

### Summary of Key Changes

The ministerial ordinance for the GPS 2026 update introduces eight macro-areas of change:

1. **New re-entry algorithm** for substitute assignment
2. **Guaranteed teaching continuity** for support roles
3. **Stricter criteria** for qualification assessment
4. **Aggregation of split hours** for more complete positions
5. **Earlier timeline** for application submission
6. **No automatic renewal** of rankings
7. **New qualification pathways** with updated requirements
8. **Strict rules** for transfers between rankings

## Re-Entry Mechanism

### The Previous Problem

In previous GPS editions, candidates who didn't express all available locations were permanently excluded from subsequent assignment rounds, even if they had high scores. This mechanism unfairly penalized many deserving teachers.

### The New Solution

The new algorithm introduces a **meritocratic re-entry system** with the following features:

- **Automatic reconsideration**: unassigned candidates are reconsidered in subsequent phases
- **Respect for preferences**: the system attempts to match emerging availabilities with expressed preferences
- **Transparency**: greater clarity on assignment criteria
- **Meritocracy**: score once again has a determining weight

### Expected Benefits

‚úÖ Reduction of unjust exclusions  
‚úÖ Better utilization of available competencies  
‚úÖ Greater fairness in assignments  
‚úÖ Transparency in selection processes
