## Key Findings Summary

### Critical Changes Identified

1. **No Automatic Renewal**
   - Must apply even to maintain current position
   - Failure to apply = 2-year exclusion

2. **24 CFU Requirement Eliminated**
   - Old qualification no longer valid
   - New 60 CFU pathways required for some categories

3. **Tight Application Window**
   - Only 20 days to submit (late Jan - mid Feb 2026)
   - Earlier than previous editions

4. **Certification Restrictions**
   - IT certifications: only Accredia-accredited entities
   - Language certifications: university-issued CLIL only
   - Potential score loss for many teachers

5. **New Re-Entry Algorithm**
   - More meritocratic assignment system
   - Automatic reconsideration in later phases
   - Respects candidate preferences

### Impact Assessment

**Benefits:**
- Greater fairness and transparency
- Reduced favoritism
- Better teaching continuity for support roles
- Higher professionalization standards

**Challenges:**
- Tight timeline creates pressure
- Many certifications may lose validity
- New pathways are costly
- Risk of technical issues with portal
